# FuoriMenu

## Overview
This Flutter application is designed for local restaurants that offer delivery services. The main goal is to help users discover nearby restaurants, view them on an interactive map, and access delivery-related information easily.

## Key Features
- Interactive map showing nearby restaurants
- Restaurant details (name, description, delivery availability)
- Location search

## Technologies
- **Flutter**: Cross-platform mobile development
- **Firebase**:
  - Firestore for restaurant and delivery data
  - Firebase Authentication

## Map Integration
The app uses a map widget to display restaurant locations based on geographic coordinates stored in Firebase. Users can tap on markers to view restaurant details.
